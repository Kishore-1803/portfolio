## 📌 Overview

This project is a **modern, responsive personal portfolio website** designed to showcase **technical projects, skills, and professional achievements** with a polished and engaging user experience.

Built using **Next.js 15 (App Router)**, **React 19**, and **TypeScript**, the portfolio emphasizes **performance, accessibility, and visual appeal**, making it ideal for recruiters, collaborators, and potential employers.

---

## 🚀 Features

- 🎨 **Interactive Background**  
  Custom particle animation system that dynamically responds to the active theme.

- 🌗 **Theme Toggle**  
  Smooth switching between **Dark** and **Light** modes with persistent user preference.

- 🖱️ **Custom Cursor**  
  Interactive dot-and-outline cursor that enhances engagement and visual feedback.

- 🗂️ **Project Showcase**  
  Dynamic project gallery with filtering and sorting for effective work presentation.

- 📱 **Fully Responsive Design**  
  Optimized for mobile, tablet, and desktop with a dedicated mobile navigation menu.

- ✉️ **Contact Form**  
  Integrated with **Formspree** for secure, serverless message handling.

## 📦 Getting Started

### Prerequisites
- Node.js **v20+** (recommended)
- npm or yarn

---

### Installation

1️⃣ **Clone the repository**
```bash
git clone https://github.com/Kishore-1803/portfolio.git
cd portfolio
````

2️⃣ **Install dependencies**

```bash
npm install
```

3️⃣ **Run the development server**

```bash
npm run dev
```

Open **[http://localhost:3000](http://localhost:3000)** in your browser to view the portfolio.

---

## 📜 Available Scripts

* `npm run dev` — Starts the development server using **Turbopack**
* `npm run build` — Creates an optimized production build
* `npm run start` — Runs the production server
* `npm run lint` — Runs ESLint to maintain code quality
